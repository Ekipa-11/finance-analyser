Podatkovni model

1. Uporabnik (User)

Atributi:

  • id (unikaten identifikator, primarni ključ)
  • username (uporabniško ime)
  • email (e-poštni naslov)
  • password_hash (šifrirano geslo)
  • created_at (datum registracije)
  • updated_at (datum zadnje posodobitve)

Opis: Entiteta Uporabnik predstavlja uporabnika aplikacije. Vsak uporabnik ima svoje podatke, ki so potrebni za prijavo in upravljanje financ.

2. Transakcija (Transaction)

Atributi:

  • id (unikaten identifikator, primarni ključ)
  • user_id (tuji ključ, povezava z uporabnikom)
  • amount (znesek transakcije)
  • date (datum transakcije)
  • category (kategorija transakcije, npr. hrana, transport, zabava)
  • description (opis transakcije, opcijsko)
  • type (tip transakcije: prihodek ali odhodek)

Opis: Entiteta Transakcija predstavlja posamezno finančno transakcijo, ki jo uporabnik vnese ali avtomatsko uvozi iz bančnega računa.

3. Proračun (Budget)

Atributi:

  • id (unikaten identifikator, primarni ključ)
  • user_id (tuji ključ, povezava z uporabnikom)
  • month (mesec proračuna)
  • year (leto proračuna)
  • income (skupni prihodki za mesec)
  • expenses (skupni odhodki za mesec)
  • savings_goal (cilj varčevanja za mesec)

Opis: Entiteta Proračun predstavlja mesečni proračun uporabnika, ki vključuje prihodke, odhodke in cilje varčevanja.

4. Kategorija (Category)

Atributi:

  • id (unikaten identifikator, primarni ključ)
  • name (ime kategorije, npr. hrana, transport, zabava)

Opis: Entiteta Kategorija predstavlja kategorije, v katere so razvrščene transakcije. Kategorije so lahko privzete ali personalizirane za vsakega uporabnika.

5. Obvestilo (Notification)

Atributi:

  • id (unikaten identifikator, primarni ključ)
  • user_id (tuji ključ, povezava z uporabnikom)
  • message (sporočilo obvestila)
  • type (tip obvestila, npr. opozorilo o presežku proračuna)
  • created_at (datum obvestila)
  • read (stanje: prebrano/neprebrano)

Opis: Entiteta Obvestilo predstavlja obvestila, ki jih uporabnik prejme od aplikacije, kot so opozorila o presežkih proračuna ali doseženih ciljih.
